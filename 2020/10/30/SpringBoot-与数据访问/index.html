整合 Spring Data JPA

Spring Data 简述

Spring 官方为了简化构建基于 Spring 框架的 数据访问技术，包含非关系型数据库、Map-Reduce 框架、云数据服务等，也包含对关系型数据库的支持。

Spring Data 官方说明

特点

​ Spring Data 提供使用统一的 API 来对数据访问层进行操作，主要由 Spring Data Commons 模块实现。Spring Data Commons 让我们在使用关系型/非关系型数据库进行数据访问时都基于 Spring 提供的统一标准。

​ 基础的 CRUD 、查询、排序、分页等操作

提供有数据访问模板类

​ MongoTemplate、RedisTemplate

JPA：Spring Data 的一种实现

  • JpaRepository 基本功能

    编写接口继承 JpaRepository 既有 crud、分页的基础功能

  • 定义符合规范的方法命名

    在接口中只需要声明符合规范的方法，即拥有对应的功能

  • @Query 自定义查询，定制查询 SQL

  • Specifications 查询（Spring Data JPA 支持 JPA 2.x 的 Criteria 查询）

任务调度：系统在某个时间执行的特定的命令或程序。

  • 系统任务

    有些重要的工作必须周而复始的执行，如病毒扫描

  • 个别用户任务

    个别用户希望执行某些程序，如数据库的备份

分区方式

mbr 分区

  • 最多支持 4 个主分区
  • 系统只能安装在主分区
  • 扩展分区要占一个主分区
  • 最大只支持 2TB，但拥有最好的兼容性

gtp 分区

  • 可以有无限个主分区
  • 最大支持 18EB 的容量

分区原理

​ 对于 Linux 来说无论有几个分区，都只有一个根目录，一个独立且唯一的文件结构，每个分区都是组成这个文件结构的一部分。

​ Linux 采用一种 挂载 的方式将分区与一个目录联系起来，要载入的分区将使它的存储空间在一个目录下获得。
